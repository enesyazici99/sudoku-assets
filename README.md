# Sudoku Assets

Map/puzzle background images for Sudoku Solitaire app.

Served via jsDelivr CDN.

## Structure

Images are organized by category:

```
sudoku-assets/
├── nature/      # Nature landscapes, forests, mountains
├── animals/     # Wildlife, pets, underwater
├── cities/      # Landmarks, architecture, skylines
├── art/         # Paintings, sculptures, artistic images
└── space/       # Galaxy, planets, nebula
```

## Usage

CDN URL format:
```
https://cdn.jsdelivr.net/gh/enesyazici99/sudoku-assets@main/{category}/{image}.webp
```

Example:
```
https://cdn.jsdelivr.net/gh/enesyazici99/sudoku-assets@main/nature/fuji.webp
https://cdn.jsdelivr.net/gh/enesyazici99/sudoku-assets@main/cities/hagiasophia.webp
https://cdn.jsdelivr.net/gh/enesyazici99/sudoku-assets@main/animals/lion.webp
```

## Categories

| Category | Description |
|----------|-------------|
| nature | Landscapes, forests, mountains, waterfalls |
| animals | Wildlife, pets, birds, underwater life |
| cities | Famous landmarks, architecture, cityscapes |
| art | Classical paintings, sculptures, artistic works |
| space | Galaxies, planets, nebulae, astronauts |
